# Channel Directory

All channels are on `irc.korin.pink:6697` (TLS required).

| Channel        | Purpose                                              |
| -------------- | ---------------------------------------------------- |
| `#stellar`     | General — music, community, off-topic                |
| `#releases`    | New release discussion and recommendations           |
| `#tech`        | Programming, hardware, Linux, audio engineering      |
| `#audiophile`  | Headphones, DACs, amps, speaker setups               |
| `#lounge`      | Slow-paced ambient chat; lower volume, no rushing    |
| `#staff`       | Staff/mod channel — invite only                      |

> Channel list may evolve. `/LIST` in your client will show the current state.

---

## Creating channels

Registered users may create channels freely. Channel ownership follows standard Ergo `ChanServ` rules — register with:

```
/MSG ChanServ REGISTER #yourchannel
```

For a channel to appear in the directory above, open an issue or bring it up in `#stellar`.
